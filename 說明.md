初始專案
1.把檔案放進建立好的github儲存庫
2.打標題說明
    【github 前綴說明】
        init:初始化專案
        feat:新增功能或畫面等
        fix:修改
        style:樣式
        doc:說明文件和相關
3.Summary標題要寫, 說明內容可寫可不寫
4.Commit -> 上傳到暫存空間
5.publish branch -> 同步到github
6.fetch orgin -> 同步成功

更新專案
1.先確認儲存庫有沒有檔案要先同步(點一下fetch origin)
2.打Summary標題(Description描述看心情)
3.Commit -> 上傳到暫存空間
4.publish origin -> 同步到github
